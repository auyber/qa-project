# QA Testing Project

Projeto de testes de software focado na prática de qualidade em aplicações web.

## 🔍 Escopo

- Testes manuais
- Casos de teste
- Gherkin (BDD)
- Bug reports
- Testes de API (Postman)
- Automação de testes (Playwright)

---

## 🎯 Objetivo

Aplicar na prática os conceitos de QA, com foco em:

- Qualidade contínua
- Prevenção de bugs
- Validação de regras de negócio
- Cobertura de cenários reais de uso

---

## ⚙️ Automação de Testes

Foi implementada uma automação básica utilizando Playwright para validar o fluxo de login.

### ✔ Cenário automatizado

- Login com credenciais válidas
- Preenchimento de campos (email e senha)
- Submissão do formulário
- Validação de redirecionamento para o Dashboard

### 🧪 Tecnologias utilizadas

- Playwright
- JavaScript

## 🧠 Abordagem de Qualidade

Este projeto segue a mentalidade de qualidade contínua, onde:

- QA atua desde o entendimento do requisito
- Testes são pensados antes da execução
- O foco é prevenir erros e não apenas encontrá-los
